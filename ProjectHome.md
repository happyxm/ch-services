## 项目介绍 ##

---

#### 项目内容 ####
```
  * 开发系统基本平台，完成一些常用的功能（如web组件等）
  * 系统权限管理模型的建立，建立一个通用的权限控制方案
  * 建立消息处理机制，完成所有系统消息统一处理。
  * 完成流程服务，主要以JBPM为参考对象
```
#### 主要技术 ####
```
  * Spring 2.5：bean、aop、template
  * Hibernate 3.2
  * JSF 2.1.9
```
